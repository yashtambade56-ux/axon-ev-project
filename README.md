# ⚡ Axon EV – Electric Vehicle Subscription Platform

Axon EV is a modern web platform designed to showcase and compare electric vehicles while offering a subscription-based rental system. The project allows users to explore EV models, compare specifications, and calculate subscription costs based on different plans.

🌐 **Live Website:** https://axonev.vercel.app/

## 📘 About the Project

Axon EV is a concept website for an **Electric Vehicle subscription service** where users can explore EV cars, compare vehicle specifications, and choose subscription plans according to their needs.

The project focuses on building a **clean, modern UI** inspired by professional automotive websites. It demonstrates front-end development using semantic HTML and structured CSS layouts.

- Car listing with images and specifications  
- Detailed vehicle information pages
